Instructions

Objective:

Avoid during 30 seconds colliding with mushroobots.

Controls:

  • Arrow Left key to move left
  • Arrow Right key to move right
  • Arrow Up key to jump

Gameplay:

  • You start the game with three lives.
  • Avoid colliding with mushroobots, as each collision will cost you one life.
  • If you lose all three lives, the game is over.
  • The game has a 30-second timer. You should hold tight.

Remember, practice makes perfect! Good luck and have fun playing Technobots - Skies of Steel.
